# Spectral-distortion-correction-based-on-CDTE-detector
本项目构建了以动态光子事件流为核心载体的物理一致性端到端仿真框架。该框架基于 Geant4 的蒙特卡洛模拟，对光子相互作用、电荷输运/极化以及电子学堆积/死时间等关键环节进行统一建模。在不同电流工况下，入射源谱变化主要以通量缩放近似表征。基于此，本文生成了覆盖 20–50 kV 管电压与 10–200 μA 管电流的多工况合成能谱数据集，为校正模型提供统一的数据基础。为同时抑制全局轮廓退化并恢复局部峰形结构，本文提出引入工况条件的双分支光谱校正深度学习模型。该模型通过趋势分支与高频细分支协同建模，实现对能谱低频背景与高频细节的解耦重建，并将管电压与管电流作为条件嵌入注入网络，以提升跨工况自适应能力。配合兼顾全谱一致性、峰形保真度及物理量级约束的复合损失函数，进一步增强了校正结果的物理一致性与训练稳定性。
